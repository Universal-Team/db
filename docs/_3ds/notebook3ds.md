This is a simple note taking/drawing app. Made it for myself to take notes during classes, but thought someone might find it useful too.
Supported features:

- Thickness (text size for text)
- Tools
- - Free drawing
- - Straight line
- - Filled rect
- - Hollow rect
- - Text
- Colors
- Saving/loading
- Undo/redo
- Multiple pages, topics, folders
