update_notes: '<h1 dir="auto">Version 3.3.0 out now!</h1>

  <h2 dir="auto">New Features:</h2>

  <h3 dir="auto">Foliage Season Editor</h3>

  <p dir="auto">Allows you to edit the season of bushes and trees. The top screen
  displays blooming/flowering information for each bush and tree.<br>

  This feature also makes use of the new keyboard slider!</p>

  <h3 dir="auto">Keyboard Slider</h3>

  <p dir="auto">A new keyboard type featuring an intuitive slider (similar to a volume
  slider) that allows you to easily set values within a specific range.<br>

  This is now used in the following cheats:</p>

  <ul dir="auto">

  <li>All Money Codes</li>

  <li>Coordinate Control Speed Settings</li>

  <li>Foliage Season Editor</li>

  <li>Time Travel</li>

  <li>Size Codes</li>

  </ul>

  <h3 dir="auto">Save Reminders</h3>

  <p dir="auto">Displays a reminder message every 25 minutes, reminding you to save
  your game.<br>

  You can adjust this interval in the Plugin Settings.</p>

  <p dir="auto">This feature is enabled by default but can be disabled in the Plugin
  Settings (below “Change Plugin Language” &amp; “Edit Menu Colors”).</p>

  <h3 dir="auto">Auto Save Backups</h3>

  <p dir="auto">The plugin now automatically backs up your entire save (all files
  such as garden_plus.dat, exhibition.dat, etc.) each time the game is launched.</p>

  <p dir="auto">A maximum of 3 backups are stored at once. If exceeded, the oldest
  backup is replaced.</p>

  <p dir="auto">Backups are saved directly to the <a href="https://github.com/BernardoGiordano/Checkpoint">Checkpoint</a>
  save directory, allowing you to restore them easily using the <a href="https://github.com/BernardoGiordano/Checkpoint">Checkpoint</a>
  app.</p>

  <p dir="auto">Enabled by default, but can be disabled in Tools -&gt; Settings.</p>

  <h3 dir="auto">Anti Scam Popup Message</h3>

  <p dir="auto">When opening the plugin for the first time after an update, a popup
  message will inform users that the plugin is free and open-source.<br>

  This message only appears once per update.</p>

  <h3 dir="auto">New License</h3>

  <p dir="auto">The plugin now uses the <a href="https://polyformproject.org/licenses/noncommercial/1.0.0"
  rel="nofollow">PolyForm Noncommercial License 1.0.0</a>, which prohibits selling
  copies of this software.</p>

  <h3 dir="auto">Custom QWERTY Keyboard (Experimental)</h3>

  <p dir="auto">Replaces the CTRPF QWERTY keyboard with the built-in Animal Crossing:
  New Leaf keyboard.</p>

  <p dir="auto">When input is required, the plugin temporarily closes, opens the New
  Leaf keyboard, and waits for the user to submit their input.</p>

  <p dir="auto">This feature is experimental and may not work perfectly.</p>

  <p dir="auto">Since the CTRPF keyboard does not support foreign characters (such
  as Chinese or Korean), this serves as a workaround.</p>

  <p dir="auto">Can be enabled in Tools -&gt; Settings.</p>

  <h3 dir="auto">Save Backup Reworked</h3>

  <p dir="auto">The Save Backup Manager has been reworked.</p>

  <p dir="auto">It now saves all files directly from disk instead of RAM. This means
  that instead of only saving garden_plus.dat, it now saves all relevant files (similar
  to <a href="https://github.com/BernardoGiordano/Checkpoint">Checkpoint</a>).</p>

  <p dir="auto">Backups are also stored in the Checkpoint save directory for easy
  restoration.</p>

  <p dir="auto">Manual restore functionality in the Save Backup Manager has been removed!
  Users are encouraged to use <a href="https://github.com/BernardoGiordano/Checkpoint">Checkpoint</a>
  instead to restore their save files.</p>

  <h3 dir="auto">Hide Design Outfits</h3>

  <p dir="auto">Based on user feedback, this cheat hides design outfits from the inventory.</p>

  <p dir="auto">If you are wearing a custom design outfit and switch to another one,
  the design outfit will no longer appear in your inventory.</p>

  <p dir="auto">This cheat is located under Inventory Codes.</p>

  <h3 dir="auto">Keep Connection In Menu (Experimental)</h3>

  <p dir="auto">Allows the plugin menu to remain open while playing online without
  losing the internet connection.</p>

  <p dir="auto">This feature is experimental and may cause issues.</p>

  <ul dir="auto">

  <li>Works well on the online island</li>

  <li>Appears stable in online towns</li>

  <li>Does not work in local multiplayer and may cause crashes</li>

  </ul>

  <p dir="auto">To prevent blocking the island, the plugin detects when players join
  or leave and automatically closes to allow the player to properly join or leave.
  (Thanks to <a class="user-mention notranslate" data-hovercard-type="user" data-hovercard-url="/users/Pienco/hovercard"
  data-octo-click="hovercard-link-click" data-octo-dimensions="link_type:self" href="https://github.com/Pienco">@Pienco</a>
  for this part)</p>

  <h3 dir="auto">Plugin now supports all standard 3DS system languages!</h3>

  <p dir="auto">We added support for:</p>

  <ul dir="auto">

  <li>Dutch</li>

  <li>Russian</li>

  <li>Portuguese</li>

  </ul>

  <p dir="auto">The plugin now automatically detects the system language for new users.<br>

  It can still be changed manually in the Plugin Settings.</p>
