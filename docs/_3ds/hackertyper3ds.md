# Hackertyper3DS

A port of the legendary [hackertyper.net](https://hackertyper.net) website for the Nintendo 3DS.

## Install (FBI)

To install the app using the QR code, open FBI on your 3DS and go to "Remote Install" and "Scan QR Code".

<img src="assets/qr.png" width="200" alt="QR code — latest CIA release">

## Features

- Color switching by pressing the Select button
- Speed customization by pressing L+D-Pad Up/Down (1-50 characters per click)
- Custom text support by editing the text.txt file (sdmc:/3ds/Hackertyper3DS/text.txt)
- Both CIA and 3DSX versions available

## LLM usage

While making this project, I used as little LLM generated code as possible, relying more on documentation. However, in the making of the publishing workflow, I did use Claude Code to troubleshoot, as I'm not really experienced with writing those workflows. I also used it to fix some bugs, but made sure that I didn't do that too often and understanding each and every line that it wrote.
